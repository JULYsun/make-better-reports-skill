# Make Better Reports

将模型输出内容/文本/文件内容转换为精美的、零依赖的单页 HTML 文档。


## 特性

- 固定侧边栏导航（左侧或右侧），支持固定/浮动切换
- 根据标题自动生成目录
- 实时搜索，支持上下翻找匹配项
- 滚动时高亮当前章节
- 代码语法高亮
- 12 种精选样式预设，覆盖 4 种风格类别
- 响应式设计（手机、平板、桌面）
- 可选数据图表（Chart.js）
- 深色/浅色主题切换

## 样式预设

| 类别 | 预设 |
|------|------|
| 专业/技术 | Modern Clean、Dark Professional、Swiss Docs |
| 现代/简约 | Minimal Light、Paper Docs、Clean Slate |
| 温暖/友好 | Warm Editorial、Notebook Style、Soft Cream |
| 开发者/终端 | Terminal Green、Code Dark、Hacker Theme |

## 文件说明

| 文件 | 用途 |
|------|------|
| `SKILL.md` | 主流程指令与各阶段说明 |
| `STYLE_PRESETS.md` | 样式定义与 CSS 变量 |
| `animation-patterns.md` | 滚动动画、交互组件、图表动效 |
| `html-template.md` | HTML 结构、JS 类、导航模式 |

## 使用方式

1. 提供你的 Markdown 内容或主题
2. 选择一种样式（或查看预览后再选）
3. 获得一个可直接打开的 `.html` 文件

---

Generate beautiful, zero-dependency single-page HTML documentation from any content.

## What It Does

Turns your LLM outputs or text into a polished, self-contained HTML documentation page — no npm, no build tools, no external dependencies. Everything runs in the browser.

## Features

- Fixed sidebar navigation (left or right) with pin/unpin support
- Auto-generated table of contents from headings
- Real-time search with match navigation
- Active section highlighting as you scroll
- Code syntax highlighting
- 12 curated style presets across 4 aesthetic categories
- Responsive design (mobile, tablet, desktop)
- Optional data charts via Chart.js
- Dark/light theme toggle

## Style Presets

| Category | Presets |
|----------|---------|
| Professional/Technical | Modern Clean, Dark Professional, Swiss Docs |
| Modern/Minimal | Minimal Light, Paper Docs, Clean Slate |
| Warm/Friendly | Warm Editorial, Notebook Style, Soft Cream |
| Developer/Terminal | Terminal Green, Code Dark, Hacker Theme |

## Files

| File | Purpose |
|------|---------|
| `SKILL.md` | Main skill instructions and workflow phases |
| `STYLE_PRESETS.md` | Visual style definitions and CSS variables |
| `animation-patterns.md` | Scroll reveals, interactive components, chart animations |
| `html-template.md` | HTML architecture, JS classes, navigation patterns |

## How to Use

1. Provide your Markdown content or topic
2. Choose a style (or get previews to pick from)
3. Get a ready-to-open `.html` file
